# robot-dog-design
Initial mechanical design of a quadruped robot dog, including body structure, leg joints, degrees of freedom, torque analysis, stability, and proposed walking gait.


## Project Overview

This project presents a simple mechanical design of a quadruped robot dog using Autodesk Fusion.

The design focuses on:

- Body structure
- Leg design
- Joints
- Degrees of freedom

## Tools Used

- Autodesk Fusion
- GitHub

## Main Design

The robot consists of:

- 1 main body
- 4 upper legs
- 4 lower legs
- 4 feet

Each leg has:

- 1 Hip Joint
- 1 Knee Joint

## Conclusion

This project demonstrates a simple initial mechanical design for a quadruped robot dog with basic joints.
